# Unit

The unit of the cost. Can be CREDITS or DOLLARS. Note: DOLLARS unit only supports PAYG plan.

## Example Usage

```typescript
import { Unit } from "@leonardo-ai/sdk/sdk/models/shared";

let value: Unit = Unit.Dollars;
```

## Values

| Name      | Value     |
| --------- | --------- |
| `Credits` | CREDITS   |
| `Dollars` | DOLLARS   |
