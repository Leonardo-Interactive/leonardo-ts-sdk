# NodeInput

A node input object for customizing a Blueprint Execution

## Example Usage

```typescript
import { NodeInput, SettingName } from "@leonardo-ai/sdk/sdk/models/shared";

let value: NodeInput = {
  nodeId: "a1b2c3d4-e5f6-7890-abcd-ef1234567890",
  settingName: SettingName.Text,
  value: "A futuristic cityscape at sunset",
};
```
