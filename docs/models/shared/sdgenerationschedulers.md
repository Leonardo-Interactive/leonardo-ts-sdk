# SdGenerationSchedulers

The scheduler to generate images with. Defaults to EULER_DISCRETE if not specified.


## Values

| Name                     | Value                    |
| ------------------------ | ------------------------ |
| `Klms`                   | KLMS                     |
| `EulerAncestralDiscrete` | EULER_ANCESTRAL_DISCRETE |
| `EulerDiscrete`          | EULER_DISCRETE           |
| `Ddim`                   | DDIM                     |
| `DpmSolver`              | DPM_SOLVER               |
| `Pndm`                   | PNDM                     |
